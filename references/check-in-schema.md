# Check-In Schema

Use this structure for weekly review.

- sessions completed (out of target)
- average energy (1-10)
- soreness notes (location + severity)
- stress level (low/medium/high)
- sleep quality (poor/fair/good)
- cycle-related readiness notes (if relevant)
- exercise performance trend (up/same/down)
- body metrics user tracks (optional):
  - weekly weight average
  - waist
  - progress photo confidence rating (1-10)

## Adjustment Rules

- If adherence high + recovery good + performance improving:
  - progress 1-2 key lifts (small load or reps)
- If adherence high + recovery poor:
  - keep intensity moderate and reduce volume 15-30%
- If adherence low due to schedule:
  - simplify to 2-3 highest-value movements/session
- If confidence low:
  - shorten sessions + increase completion wins
